Origem da Internet

A Internet nasceu a partir da ARPANET nos anos 60, um projeto militar norte-americano. Nos anos 80 e 90, foi expandida para uso acadêmico e comercial. O cientista Tim Berners-Lee criou a World Wide Web em 1991, que permitiu acesso a páginas interligadas. Atualmente, a Internet é indispensável para comunicação, trabalho e lazer. Referência: Wikipédia - História da Internet

O que são Navegadores (Browsers)?

Um browser é o programa responsável por interpretar os códigos HTML, CSS e JavaScript e exibir páginas para o usuário. Exemplos: Google Chrome, Mozilla Firefox, Microsoft Edge e Safari. Sem um navegador, não conseguimos visualizar a Web como conhecemos. Referência: MDN Web Docs - Browsers

O que são Servidores Web?

Um servidor web é o software ou hardware responsável por armazenar, processar e entregar páginas da Internet aos navegadores. Exemplos de servidores: Apache, Nginx, IIS. Sem servidores, os sites não poderiam ser acessados. O servidor é o "anfitrião" e o navegador é o "visitante". Referência: W3Schools - Web Server

Estrutura básica de uma página HTML

Toda página HTML segue uma estrutura padrão:

  • !DOCTYPE html → define o tipo de documento.
  • html → envolve todo o conteúdo.
  • head → contém título, metadados e links para CSS/JS.
  • body → contém o conteúdo visível (textos, imagens, links).
  • HTML significa HyperText Markup Language.

Diferença entre listas ordenadas e não ordenadas

Listas ordenadas (ol) apresentam itens numerados ou sequenciais. Listas não ordenadas (ul) apresentam itens sem ordem definida, apenas marcadores.

Exemplo Prático

    Lista ordenada (etapas para preparar um café):

  1. Esquentar a água.
  2. Colocar o pó no filtro.
  3. Despejar a água quente.
  4. Servir.

    Lista não ordenada (compras de mercado):

  • Café
  • Açúcar
  • Arroz
  • Macarão

Conclusão: listas ordenadas seguem uma sequência; listas não ordenadas não dependem de ordem.
